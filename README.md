# Russian Fonts for English Call of Duty 4 (CoD4x Compatible)

Добавляет поддержку кириллицы в **английскую версию Call of Duty 4: Modern Warfare**.

> [!IMPORTANT]
> После выхода **CoD4x 20.0**:
> - Проверка IWD файлов ломала этот метод.
> - Игроков кикало с серверов.

## 📌 Описание

В стандартной английской версии игры отсутствует поддержка кириллицы.  
Этот мод исправляет это и позволяет:
- Писать на русском в чате.
- Видеть кириллицу без установки полной русской локализации.
- Сохранить оригинальные английские шрифты (без "огромных" русских).

💡 Особенно полезно для игроков, которым не нравится русская локализация, но нужна поддержка русского языка.

## 📥 Установка

**Скачайте архив:** [Russian chat for english version.zip](https://github.com/wxndrxus/cod4-russian-fonts/releases/download/1.0/cod4-russian-fonts-1.0.zip)

- **Файл: localized_code_post_gfx_mp.ff**
- - Перейдите в папку `zone/english`.
- - Замените файл.

- **Файл: gamefonts_pc.iwi**
- - Откройте: main/localized_english_iw00.iwd с помощью **7-Zip**.
- - Перейдите в папку **images**.
- - Замените файл.

> [!NOTE]
> Если у вас по какой-то причине не отображаются шрифты, введите команду **/ui_extrabigfont** от 0 до 1, (например 0.4 или 0.6)

## ❓ FAQ
